# x-anim

This is a POC of an interactive-image project.


## Usage

Move the mouse through the window to see the movement.

Keyboard usage:

- `NumPad+  or  W`: Increase the factor
- `NumPad-  or  S`: Decrease the factor

The current factor may be visualized in the bottom-right corner.
